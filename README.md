# 🛒 Supermarket Dashboard – Power BI Project

## 🚀 Project Overview

This project is an interactive **Supermarket Sales Dashboard** developed using **Microsoft Power BI** to analyze supermarket sales performance, customer purchasing behavior, profit trends, and operational insights.

The dashboard provides a complete business overview and helps stakeholders make data-driven decisions using real-time analytics and visual storytelling.

The dashboard helps businesses analyze:

* Total sales and revenue trends
* Profitability analysis
* Product category performance
* Customer purchasing behavior
* Branch and regional performance
* Payment method insights
* Monthly and yearly sales trends
* KPI monitoring and forecasting

# 📑 Dataset Information

The dataset contains important supermarket business data including:

| Column Name    | Description                   |
| -------------- | ----------------------------- |
| Invoice ID     | Unique transaction identifier |
| Branch         | Supermarket branch            |
| City           | Sales location                |
| Customer Type  | Member/Normal customer type   |
| Gender         | Customer gender               |
| Product Line   | Product category              |
| Unit Price     | Product selling price         |
| Quantity       | Number of items sold          |
| Tax            | Tax amount                    |
| Total Sales    | Total bill amount             |
| Date           | Transaction date              |
| Time           | Purchase time                 |
| Payment Method | Mode of payment               |
| COGS           | Cost of goods sold            |
| Gross Income   | Profit earned                 |
| Rating         | Customer satisfaction rating  |
